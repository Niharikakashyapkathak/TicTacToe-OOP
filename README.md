# 🎮 Tic Tac Toe (C++ OOP)

A simple Tic Tac Toe game built using Object-Oriented Programming in C++.

## ✨ Features
- 2 Player Game
- Win detection logic
- Clean modular code (Board, Player, Game)

## ▶️ How to Run

g++ main.cpp Game.cpp Board.cpp Player.cpp -o main  
./main
